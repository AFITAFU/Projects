CAREER STARTER - Build Your Foundation

This course is good for students who want a cushioned introduction to a career in I.T and the basics of foundational programming languages and tools.

The Career Starter courses are designed for young students who are either in college or have recently graduated and want to explore the I.T. industry to kick start their careers. Kick things off with a solid foundation in web coding languages, and industry best practices.

  • Introduction to HTML5
  • CSS3
  • Responsive Websites Using Bootstrap
  • Git & GitHub
  • Milestone Project – *Resume/Portfolio Website Design (HTML5)
  • Duration: Instructional hours – 36, Lab hours – 24

By the end of this course you will an in-depth understanding of basic web design and version control, and leave with a built-from-scratch personal resume/portfolio website to use on job search. It will include three professional quality web projects to show off.

CAREER BUILDER – Get Your Hands Dirty

Codebits Builder is a course package for intermediate candidates who already have some I.T starter skills. It was designed to equip the candidates with in-depth knowledge in the in-demand front-end technologies like Javascript, jQuery, Angular, etc. The curriculum ensures that you get hands-on training experience through challenges and milestone projects.

  • Javascript
  • Advanced HTML5
  • Angular
  • Milestone Project – *Job Listing Application (Angular)
  • Duration: Instructional hours – 48, Lab hours – 32
